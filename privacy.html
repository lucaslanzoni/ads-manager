Política de Privacidade

Última atualização: abril de 2026

1. Sobre este aplicativo

O Ads Manager é uma ferramenta de uso interno para criação e publicação de campanhas de anúncios no Meta Ads. O aplicativo não é disponibilizado ao público geral.

2. Dados coletados

O aplicativo acessa dados do Meta (Facebook/Instagram) exclusivamente para criar e gerenciar campanhas publicitárias na conta de anúncios autorizada pelo usuário. Nenhum dado pessoal de terceiros é coletado ou armazenado.

3. Uso dos dados

Os dados de acesso (tokens) são utilizados apenas para comunicação com a API de Marketing do Meta. Não são compartilhados com terceiros nem utilizados para fins além da publicação de anúncios.

4. Armazenamento

Nenhum dado pessoal é armazenado de forma permanente. Tokens de acesso são mantidos como variáveis de ambiente no servidor e não são expostos ao público.
